# Temperature Converter Website

## Overview

This project is a Temperature Converter Website developed as part of the Oasis Infobyte Web Development & Designing Internship (Level 1).

The application allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin while validating user input and handling edge cases such as temperatures below absolute zero.

## Features

* Celsius to Fahrenheit & Kelvin Conversion
* Fahrenheit to Celsius & Kelvin Conversion
* Kelvin to Celsius & Fahrenheit Conversion
* Input Validation
* Absolute Zero Validation
* Responsive User Interface
* User-Friendly Design

## Technologies Used

* HTML5
* CSS3
* JavaScript

│
