# BMI Calculator

## Overview

BMI Calculator is a Python application that calculates a user's Body Mass Index (BMI) using their weight and height. It classifies the result into standard health categories such as Underweight, Normal, Overweight, and Obese.

## Features

* Calculate BMI using weight and height
* Health category classification
* Input validation
* User-friendly interface
* Store BMI records in CSV format

## Technologies Used

* Python
* CSV Module

## Formula Used

BMI = Weight (kg) / Height² (m²)

## Project Structure

```text

├── bmi.py
├── bmi_data.csv
└── README.md
```
